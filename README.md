# Tagentdecker MVP

## Vision
„Tagentdecker“ liefert perfekt geplante Tagesausflüge für 25–40 Jährige in Mainfranken.

## MVP Features
- Standortbasierte Vorschläge
- Zeitbudget Auswahl (3h / 5h / 8h)
- Interessenfilter (Natur, Kultur, Gastro)
- Anzeige von 1–3 fertigen Touren
- Kartenlink zu jedem Stop
- Mobile-first Design

## Erste Touren Beispiele
- Tour 1: Würzburg Altstadt + Marktplatz + Käppele 
- Tour 2: Rhön Naturwanderung + Sommerrodelbahn Wasserkuppe
- Tour 3: Main-Radweg + Mittagessen in Zeil am Main + Hexen-Turm
